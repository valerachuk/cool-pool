# Answering a pool

  1. Open your shared link.
  2. Answer all questions.
  3. Press the SAVE button below the pool.

# Signing in

  1. Press Sign in button on header.
  2. In the opened window select your google account and press allow/continue.
  3. You will be redirected to the pool list page.

# Signing out

To sign out press the Sign out button on the header.

# Creating a pool

  1. Navigate to the pool list page.
  2. Press NEW POOL button, you will be redirected to the pool creation page.
  3. Fill all required fields and press the SAVE button.
  4. After a few seconds your pool will be saved.

TIP

  • To add a question block to your pool press big plus ➕ button below the latest question block in the pool.
  • To duplicate a question block press copy button 📋 below at the bottom right corner of it.
  • To delete a question block press delete button 🗑 below at the bottom right corner of it.
  • To add an option to the question block, press the small plus ➕ button at the bottom left corner of it.
  • To change type exactly one / at least one answer of a question block, use select at the top right corner.

# Deleting a pool

  1. Navigate to the pool list page.
  2. Press button with trash icon 🗑 on corresponding row.
  3. Confirm deleting by pressing the AGREE button on the appeared window.

# Sharing a pool

  1. Navigate to the pool list page.
  2. Press button with chain icon 🔗 on corresponding row.
  3. Link will be copied to a clipboard.

# Viewing pool results

  1. Navigate to the pool list page.
  2. Press button with eye icon 👁 on corresponding row.
